✅ 결재 승인 기준
· 외화 $100 이상 또는 한화 10만원 이상은 팀장 결재 필요
· 기준 금액 이상 시 해당 팀장님께 승인요청 메일 자동 발송
· 팀원 → 각 팀장 → 담당 임원
· 대한항공 → 박원욱 대표
📅 결재 일정
· 신청 마감: 매주 화요일 오후 5시 이전 신청 + 팀장 승인 완료
· 결재 진행: 매주 수요일 업무시간 중 순차 진행
· 보통 수요일 낮까지 완료 (상황에 따라 지연 가능)
· 수요일 외 결제 필요 시 사전에 김지혜 차장 연락 필수 (대한항공 제외)
🔔 결재 결과 알림
· 결제 완료 시: 신청자에게 완료 메일 자동 발송
· 반려 시: 신청자에게 반려 메일 자동 발송 (사유는 팀장 확인)
· 수정 필요 시: 신청서 재작성 → 마지막에 '재신청' 선택 + 사유 작성
🔍 결재 현황 확인
승인대기 / 승인 / 반려 / 결제완료 실시간 확인 가능 (조회 전용)
